# Color Struct

Color Struct = "\033<`ESC String`>[0;<`Style`> 31m<`Color`>"
\e = \033 Short other element apply
\x18 = 16bit
\u001B = UniCode

```sh
## ESC Color
RESET="\033[0; 0m"
BLACK="\033[0; 30m"
RED="\033[0; 31m"
GREEN="\033[0; 32m"
YELLOW="\033[0; 33m"
BLUE="\033[0; 34m"
PINK="\033[0; 35m"
CYAN="\033[0; 36m"
WHITE="\033[0; 37m"
NORMAL="\033[0; 39m"
```
